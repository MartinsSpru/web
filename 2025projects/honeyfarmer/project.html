How to play:

Upgrade unlocking:

Every upgrade has to be unlocked, if you meet the requirements after pressing the unlock button then you will gain access to the item, you can try this out with the first item of the production upgrades.

Getting honey:

You get some honey every second, if you want to increase the amount of honey you can go and buy production upgrades.

Getting money:

To get money you have to unlock markets, you can find those on the sell page.

Saving:

To save go to the save page and press 'create a save' then youl get an alert with your code, copy it and save it somewhere.

Loading a save:

To load a save you have to put your save code into the loading input field in the save page and then press load save.
